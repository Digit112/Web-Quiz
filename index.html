Learn Japanese

Help | Writeup

Select from the many kinds of questions below to start learning Japanese! All changes will take effect immediately. The quiz tracks your progress and may change somewhat over time. This information can be reset by clicking "Reset Progress". Click "Next Question" when you are ready to begin.

If you know nothing at all about the Japanese language, you can read the Help page for a quick starter. To be clear, you are expected to guess randomly at first. You'll pick it up pretty fast. I recommend selecting All Questions -> Hiragana -> Basic Hiragana (in the menu below) to start, and leaving the settings at their defaults.

Note: This is a work in progress! It requires minor tuning and probably some bugfixing, but more than anything it requires more questions. I have made this site live before it was finished only so that I could use it while I'm away from home.

Random Adaptive Quiz

Questions that you frequently answer incorrectly will appear more often. The same question cannot appear twice in a row.

Use Question Window

A small number of questions will be available to begin. As you master old questions, the pool of questions will increase in size.

Shuffle Questions

Questions will become available in a predefined order. Only works if the question window is enabled, otherwise all questions become available immediately.
